# 执行install.sh脚本功能

## 执行方案

由于当前环境是Windows，bash脚本无法直接执行，我将手动执行脚本中的命令来完成相同的功能：

### 1. 清理项目

```bash
./gradlew clean
```

### 2. 构建Debug版本

```bash
./gradlew assembleDebug
```

### 3. 安装到设备

```bash
./gradlew installDebug
```

### 4. 启动应用

```bash
adb shell am start -n com.gradle.aicodeapp/com.gradle.aicodeapp.MainActivity
```

## 预期效果

* 项目将被清理和重新构建

* 应用将被安装到连接的Android设备上

* 应用将自动启动并打开

## 注意事项

* 需要确保有Android设备连接到电脑

* 需要确保adb命令可用

* 需要确保设备已启用USB调试模式

* 执行过程中可能需要在设备上授权安装应用
